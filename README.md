## frases da sorte app

Neste projeto foi utilizado um conceito importante de lógica, onde o objetivo foi montar uma estrutura condicional.
O app consiste em receber do usuário sua data de aniversário e devolver uma frase da sorte de acordo com a data escolhida.Se por acaso o usuário escolher uma data fora do padrão do calendário, será exibida a mensagem "DIA IVÁLIDO", conscientizando-o a escolher a data dentro do padrão.
